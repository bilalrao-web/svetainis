# Professional Web Development Project - UPLANCE Agency Website

## Project Overview
Developed a complete, modern, and fully dynamic agency website with a comprehensive admin panel for content management. The project features multi-language support (Lithuanian & English), dynamic content management, and a responsive design built with cutting-edge technologies.

## Technologies & Stack

### Backend
- **Laravel 12** - Latest PHP framework
- **Livewire 3** - Full-stack framework for dynamic components
- **MySQL** - Database management
- **Eloquent ORM** - Database interactions

### Frontend
- **Tailwind CSS 4** - Modern utility-first CSS framework
- **Vite** - Next-generation frontend build tool
- **AOS (Animate On Scroll)** - Smooth scroll animations
- **Font Awesome** - Icon library
- **Responsive Design** - Mobile-first approach

## Key Features Implemented

### 1. Dynamic Admin Panel
- **Complete CRUD Operations** for all content types
- **User Authentication** with secure login/logout
- **Dashboard** with statistics and overview
- **Settings Management** - Site name, logo, contact info, social media links
- **Service Management** - Create, edit, delete services with rich content
- **Portfolio Management** - Showcase projects with detailed information
- **Menu Management** - Dynamic header and footer navigation
- **Page Management** - Custom content pages
- **Image Upload** - Logo and media management

### 2. Multi-Language Support (i18n)
- **Bilingual Website** - Lithuanian (LT) and English (EN)
- **Language Switcher** - Easy toggle between languages
- **Database Translations** - Dynamic content stored with translations
- **Session & Cookie Persistence** - Remembers user's language preference
- **Translated Accessors** - Automatic translation retrieval based on locale

### 3. Frontend Features
- **Landing Page** - Hero section with services showcase
- **Services Page** - Grid layout with service cards
- **Service Detail Pages** - Rich content pages with structured information
- **Portfolio Page** - Modern card-based layout

- **Contact Page** - Contact form with validation
- **Responsive Header** - Dynamic menu with active state highlighting
- **Footer** - Four-column layout with navigation, contact, social media, and about sections

### 4. Content Management Features
- **Dynamic Services** - Manage service titles, descriptions, intros, benefits, and more
- **Portfolio Projects** - Showcase projects with technologies, challenges, solutions, and results
- **Menu Items** - Flexible navigation with route or URL support
- **Site Settings** - Centralized configuration management

### 5. User Experience Enhancements
- **Smooth Animations** - AOS animations for engaging user experience

## Database Structure

### Tables Created
- `settings` - Site-wide configuration
- `services` - Service information with translations
- `portfolios` - Portfolio project details
- `menu_items` - Navigation menu items with translations
- `pages` - Custom content pages
- `contact_messages` - Contact form submissions
- `users` - Admin user management

### Advanced Features
- **JSON Columns** - Store complex data structures
- **Translation Columns** - Multi-language support at database level
- **Soft Deletes** - Data preservation
- **Ordering System** - Custom ordering for all content types
- **Active/Inactive Toggle** - Content visibility control
